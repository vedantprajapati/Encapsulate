# /Open
## Open a Shortcut Link

To open a shortcut, simply type `/o` or `/open` followed by a shortcut.

The following shortcuts are available
```
maps
q(quercus)
gpt
bank
yt 
drive
docs
sheets
cal
```

Here are a few example commands:

```
/open maps
/o gpt
/open docs
```
